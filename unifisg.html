Unifi Security Gateway

De Ubiquiti UniFi Security Gateway is een professionele router met firewall waarmee je een klein bedrijfsnetwerk
compleet beveiligt tegen gevaren van buitenaf. De router kan tot één miljoen pakketjes per seconden versturen,
waardoor hij zeer handig is voor peer to peer netwerken of bijvoorbeeld wanneer je een website host. De UniFi Security Gateway beschikt over handige, zakelijke functies.
Zo beperk je eenvoudig toegang tot websites en zet je via VPN beveiligde verbindingen op.
